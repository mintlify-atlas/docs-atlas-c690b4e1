## Overview

The course management system allows professors to create courses, add chapters, edit content using markdown, and manage course metadata. The system features a real-time markdown editor with preview capabilities.

## Course creation workflow

Professors can create new courses through the dashboard interface.

<Steps>
  <Step title="Navigate to dashboard">
    From the professor dashboard at `pages/dashboard.php`, click the "Nouveau cours" button.
  </Step>
  
  <Step title="Initialize course">
    The system creates a new course with default values and redirects to the course editor.
  </Step>
  
  <Step title="Create initial chapter">
    A demo chapter is automatically created with a markdown file.
  </Step>
  
  <Step title="Customize course">
    Edit the course name, description, category, and cover image.
  </Step>
  
  <Step title="Add content">
    Create additional chapters and write course content using the markdown editor.
  </Step>
</Steps>

```php
if($_GET['cours'] == "00000"){ //new course

        mkdir($dir_path, 0777, true); 
    }
    
    if (!file_exists($file_path)) {
        $init_content = "# Chapitre 1 : \n \n à vous de jouer !";
        file_put_contents($file_path, $init_content); 
    }

    $sqlUpdate = "UPDATE Chapitres SET fichier_url = :url WHERE id = :id";
    $request_update = $db->prepare($sqlUpdate);
    $request_update->bindParam(":url", $file_path);
    $request_update->bindParam(":id", $chapter_id);
    $request_update->execute();

    header('Location: addCourse.php?cours=' . $idCours);
    exit();
}
```

<Info>

### Delete chapter

From `pages/addCourse.php:122-138`:

```php
function DeleteChap(){
    global $db;
    $sql = "DELETE FROM Chapitres WHERE id = :id ";
    $request = $db->prepare($sql);
    $request->bindParam(":id", $_COOKIE['activeChapId']);
    $request->execute();
    
    // Delete markdown file
    $path = "../md_files/". $_GET['cours'] . '/'. $_COOKIE['activeChapId'] . '.md';
    if (file_exists($path)) {
        if (!unlink($path)) {
            header('Location: dashboard.php');
        }
    }
    
    // Remove cookie
    setcookie("activeChapId", "", time() - 3600, "./");
    header('Location: addCourse.php?cours=' . $_GET['cours']);
    exit();
}
```

<Warning>
  Deleting a chapter permanently removes both the database record and the markdown file. This action cannot be undone.
</Warning>

### Image upload via drag and drop

The editor supports drag-and-drop image uploads from `pages/addCourse.php:428-469`:

```javascript
mdEditor.addEventListener('drop', async (e) => {
  e.preventDefault();
  mdEditor.style.border = "3px";
  const files = e.dataTransfer.files;
  if (!files.length) return;
  const file = files[0];

  if (!file.type.startsWith('image/')) {
    alert("Le fichier n'est pas une image. \n Veuillez entrer une image valide");
    return;
  }

  const reader = new FileReader();
  reader.onload = async function () {
    const base64 = reader.result; 
    const markdownImage = `![image](${base64})`;

    const savedPath = await SaveImg(markdownImage);

    if (savedPath) {
      const cursorPos = mdEditor.selectionStart;
      const before = mdEditor.value.substring(0, cursorPos);
      const after = mdEditor.value.substring(cursorPos);
      const markdownFinal = `![image](${savedPath})`;
      mdEditor.value = before + markdownFinal + after;
    } else {
      alert("Erreur lors de la sauvegarde de l'image.");
    }
  };

  reader.readAsDataURL(file);
});
```

<Note>
  Images are converted to base64 and then saved to the server, with the path inserted into the markdown.
</Note>

## Database schema

### Cours table

```sql
CREATE TABLE Cours (
    id INT PRIMARY KEY AUTO_INCREMENT,
    nom VARCHAR(255) NOT NULL,
    illustration_url TEXT,
    description TEXT,
    prof_id INT NOT NULL,
    categorie_id INT,
    date_creation DATETIME DEFAULT CURRENT_TIMESTAMP,
    FOREIGN KEY (prof_id) REFERENCES Utilisateurs(id) ON DELETE CASCADE,
    FOREIGN KEY (categorie_id) REFERENCES Categories(id)
);
```

### Chapitres table

```sql
CREATE TABLE Chapitres (
    id INT PRIMARY KEY AUTO_INCREMENT,
    titre VARCHAR(255) NOT NULL,
    cours_id INT NOT NULL,
    fichier_url TEXT,
    FOREIGN KEY (cours_id) REFERENCES Cours(id) ON DELETE CASCADE
);
```
