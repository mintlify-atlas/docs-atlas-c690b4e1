## Overview

The student enrollment system allows users to browse available courses, filter by category, view course details, and enroll in courses. The platform tracks views, enrollments, and provides personalized course recommendations.

## Course discovery workflow

Students can discover courses through multiple pathways.

<Steps>
  <Step title="Browse all courses">
    Navigate to the formations page to see all available courses.
  </Step>
  
  <Step title="Filter by category">
    Use the category dropdown to filter courses by topic area.
  </Step>
  
  <Step title="Search courses">
    Enter keywords in the search bar to find specific courses.
  </Step>
  
  <Step title="View course details">
    Click on a course card to see the full course description and syllabus.
  </Step>
  
  <Step title="Enroll in course">
    Click the enrollment button to register for the course.
  </Step>
</Steps>

### Track course views

The system automatically tracks when users view courses from `pages/detail.php:50-95`:

<Tabs>
  <Tab title="Authenticated users">
    ```php
    if (IsConnected()) {
        $sql = 'SELECT id FROM Utilisateurs WHERE id = 
                (SELECT user_id FROM sessions WHERE token = :token)';
        $request = $db->prepare($sql);
        $request->bindParam(':token', $_COOKIE['user_token']);
        $request->execute();
        $user = $request->fetch(PDO::FETCH_ASSOC);

        if ($user) {
            // Check if view already exists
            $check_vue = $db->prepare("SELECT id FROM Vues 
                                       WHERE utilisateur_id = :user_id 
                                       AND cours_id = :cours_id");
            $check_vue->bindParam(':user_id', $user['id'], PDO::PARAM_INT);
            $check_vue->bindParam(':cours_id', $id_cours, PDO::PARAM_INT);
            $check_vue->execute();

            // Create view if it doesn't exist
            if ($check_vue->rowCount() == 0) {
                $stmt_vue = $db->prepare("INSERT INTO Vues 
                                          (utilisateur_id, cours_id, date_vue) 
                                          VALUES (:user_id, :cours_id, NOW())");
                $stmt_vue->bindParam(':user_id', $user['id'], PDO::PARAM_INT);
                $stmt_vue->bindParam(':cours_id', $id_cours, PDO::PARAM_INT);
                $stmt_vue->execute();
            }
        }
    }
    ```
  </Tab>

### Process enrollment

From `pages/detail.php:118-142`:

```php
if (isset($_POST['inscription']) && IsConnected()) {
    $sql = 'SELECT id FROM Utilisateurs WHERE id = 
            (SELECT user_id FROM sessions WHERE token = :token)';
    $request = $db->prepare($sql);
    $request->bindParam(':token', $_COOKIE['user_token']);
    $request->execute();
    $user = $request->fetch(PDO::FETCH_ASSOC);

    if ($user && !$is_inscrit) {
        // Add enrollment
        $stmt_inscription = $db->prepare("INSERT INTO Inscriptions 
                                          (etudiant_id, cours_id) 
                                          VALUES (:user_id, :cours_id)");
        $stmt_inscription->bindParam(':user_id', $user['id'], PDO::PARAM_INT);
        $stmt_inscription->bindParam(':cours_id', $id_cours, PDO::PARAM_INT);
        $stmt_inscription->execute();

        // Add view entry
        $stmt_vue = $db->prepare("INSERT INTO Vues 
                                  (utilisateur_id, cours_id) 
                                  VALUES (:user_id, :cours_id)");
        $stmt_vue->bindParam(':user_id', $user['id'], PDO::PARAM_INT);
        $stmt_vue->bindParam(':cours_id', $id_cours, PDO::PARAM_INT);
        $stmt_vue->execute();

        // Redirect to avoid form resubmission
        header("Location: detail.php?id=" . $id_cours);
        exit();
    }
}
```

## Database schema

### Inscriptions table

```sql
CREATE TABLE Inscriptions (
    id INT PRIMARY KEY AUTO_INCREMENT,
    etudiant_id INT NOT NULL,
    cours_id INT NOT NULL,
    date_inscription DATETIME DEFAULT CURRENT_TIMESTAMP,
    FOREIGN KEY (etudiant_id) REFERENCES Utilisateurs(id) ON DELETE CASCADE,
    FOREIGN KEY (cours_id) REFERENCES Cours(id) ON DELETE CASCADE,
    UNIQUE KEY unique_enrollment (etudiant_id, cours_id)
);
```

### Vues table

```sql
CREATE TABLE Vues (
    id INT PRIMARY KEY AUTO_INCREMENT,
    utilisateur_id INT NULL,
    cours_id INT NOT NULL,
    ip_address VARCHAR(45) NULL,
    date_vue DATETIME DEFAULT CURRENT_TIMESTAMP,
    FOREIGN KEY (utilisateur_id) REFERENCES Utilisateurs(id) ON DELETE CASCADE,
    FOREIGN KEY (cours_id) REFERENCES Cours(id) ON DELETE CASCADE
);
```

### Favoris table

```sql
CREATE TABLE Favoris (
    id INT PRIMARY KEY AUTO_INCREMENT,
    utilisateur_id INT NOT NULL,
    cours_id INT NOT NULL,
    date_ajout DATETIME DEFAULT CURRENT_TIMESTAMP,
    FOREIGN KEY (utilisateur_id) REFERENCES Utilisateurs(id) ON DELETE CASCADE,
    FOREIGN KEY (cours_id) REFERENCES Cours(id) ON DELETE CASCADE,
    UNIQUE KEY unique_favorite (utilisateur_id, cours_id)
);
```

## User experience flows

<Steps>
  <Step title="Guest user browsing">
    Anonymous users can browse courses, search, filter by category, and view course details. Views are tracked by IP address.
  </Step>
  
  <Step title="Registration prompt">
    When attempting to enroll, guests are redirected to the connection page to create an account.
  </Step>
  
  <Step title="Student enrollment">
    Logged-in students can enroll in courses with a single click. The enrollment is recorded immediately.
  </Step>
  
  <Step title="Course access">
    After enrollment, students can access the course viewer to read chapters and track progress.
  </Step>
</Steps>

<Warning>
  Students cannot enroll in courses that have no chapters. Professors must add at least one chapter before students can enroll.
</Warning>
