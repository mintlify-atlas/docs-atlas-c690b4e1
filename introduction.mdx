## What is SkillUp?

SkillUp is a free online learning platform designed to help users train and find information across multiple disciplines. Developed as part of web and database courses at the **Université de Tours** for first-year computer science students, the platform focuses primarily on computer science education while being architected to support diverse domains.

The platform enables learners to study at their own pace with access to comprehensive courses, while providing instructors with powerful tools to create and manage educational content.

## Why SkillUp?

SkillUp was built with three core principles in mind:

### 100% free learning

All courses and resources are completely free. The platform provides full access to educational materials without any cost, enabling anyone to learn and develop their skills without financial barriers.

### Self-paced education

Users can organize their learning according to their own schedule, without time pressure or rigid constraints. The flexible architecture supports individual learning styles and paces.

### Always up-to-date content

Courses are continuously updated to stay current with the latest knowledge and skills in each domain, ensuring learners have access to relevant and modern educational content.

## Key statistics

SkillUp tracks platform engagement through real-time statistics:

```php ~/workspace/source/include/stats.php
function getStats() {
    $db = connectDB();
    $stats = [
        'utilisateurs' => 0,
        'cours' => 0,
        'profs' => 0
    ];
    
    $sql = "SELECT COUNT(*) as total FROM Utilisateurs";
    $query = $db->query($sql);
    $stats['utilisateurs'] = $query->fetch(PDO::FETCH_ASSOC)['total'];
    
    $sql = "SELECT COUNT(*) as total FROM Cours";
    $query = $db->query($sql);
    $stats['cours'] = $query->fetch(PDO::FETCH_ASSOC)['total'];
    
    $sql = "SELECT COUNT(*) as total FROM Utilisateurs WHERE role = 'professeur'";
    $query = $db->query($sql);
    $stats['profs'] = $query->fetch(PDO::FETCH_ASSOC)['total'];
    
    return $stats;
}
```

The homepage displays these statistics to showcase platform activity and engagement.

## Technology stack

SkillUp is built using:

- **Backend**: PHP 8.0+ with PDO for database interactions
- **Database**: MySQL 8.0+ with utf8mb4 character set
- **Frontend**: Vanilla JavaScript with responsive CSS
- **Architecture**: MVC-style organization with session management
- **Security**: Password hashing, prepared statements, and CSRF protection

## Project structure

The codebase follows a clear organizational structure:

```
/api          - REST API endpoints for course management
/assets       - Static resources (CSS, JS, images, SQL)
/include      - PHP includes (config, database, utilities)
/pages        - Application pages (profile, courses, dashboard)
index.php     - Landing page and entry point
```

## Academic origins

Developed at the **Université de Tours** as part of first-year computer science curriculum, SkillUp serves as both a practical learning platform and a demonstration of modern web development practices. The project encompasses web development fundamentals, database design, and user experience principles.
