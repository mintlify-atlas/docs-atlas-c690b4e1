## Overview

SkillUp provides a complete learning management system with features for students, instructors, and administrators. The platform supports course creation, content management, user progress tracking, and API-based integrations.

## User management

### Authentication and sessions

SkillUp implements secure session management with token-based authentication:

```php ~/workspace/source/include/sessionManager.php
function IsConnected(): bool {
    $db = connectDB();

    // Delete expired sessions on each call
    $now = new DateTime();
    $nowFormatted = $now->format('Y-m-d H:i:s');

    $sql = "DELETE FROM sessions WHERE expires_at < :now";
    $stmt = $db->prepare($sql);
    $stmt->bindParam(':now', $nowFormatted);
    $stmt->execute();

    if (!isset($_COOKIE['user_token'])) {
        return false; 
    }
    
    $sql = "SELECT user_id, expires_at FROM sessions WHERE token = :token";
    $stmt = $db->prepare($sql);
    $stmt->bindParam(':token', $_COOKIE['user_token'], PDO::PARAM_STR);
    $stmt->execute();
    $result = $stmt->fetch(PDO::FETCH_ASSOC);

    if (!$result) {
        return false;
    }

    $expires_at = new DateTime($result['expires_at']);
    if ($expires_at > $now) {
        return true;  
    }

    return false; 
}
```

The session management automatically cleans up expired sessions and validates tokens on every request.

## Trending courses

The homepage features trending courses based on enrollment:

```php ~/workspace/source/include/stats.php
function getTrendingCourses($limit = 4) {
    $db = connectDB();
    
    $sql = "SELECT c.id, c.nom, c.illustration_url, c.description, cat.nom as categorie_nom, 
            COUNT(i.id) as nombre_inscrits
            FROM Cours c
            LEFT JOIN Inscriptions i ON c.id = i.cours_id
            LEFT JOIN Categories cat ON c.categorie_id = cat.id
            GROUP BY c.id
            ORDER BY nombre_inscrits DESC, c.date_creation DESC
            LIMIT :limit";
    
    $stmt = $db->prepare($sql);
    $stmt->bindParam(':limit', $limit, PDO::PARAM_INT);
    $stmt->execute();
    return $stmt->fetchAll(PDO::FETCH_ASSOC);
}
```

```bash
curl -X POST https://skillup.great-site.net/api/SaveMd.php \
  -H "Content-Type: application/x-www-form-urlencoded" \
  -d "token=your_api_key" \
  -d "chapter=13" \
  -d "cours=12" \
  --data-urlencode "markdown=# Chapitre 1 :"
```

## Database architecture

SkillUp uses a well-structured MySQL database:

### Core tables

- **Utilisateurs**: User accounts with roles and authentication
- **Cours**: Course information and metadata
- **Chapitres**: Individual course chapters with Markdown content
- **Categories**: Course categorization
- **Inscriptions**: Course enrollments
- **Vues**: Course view tracking
- **Favoris**: User favorites/likes
- **Sessions**: Active user sessions
- **KeyTable**: API authentication keys
- **DemandeProf**: Professor role requests
- **ApiLogs**: API request logging
